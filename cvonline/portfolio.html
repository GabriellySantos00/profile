GameOn

Descrição

O GameOn é um protótipo de interface desenvolvido no Figma como projeto de finalização do primeiro semestre do curso de Análise e Desenvolvimento de Sistemas, em parceria com minha colega Camilly. A proposta foi criar uma plataforma voltada para a organização e participação em eventos de eSports, reunindo recursos para jogadores, equipes e organizadores.

Tecnologias utilizadas

  • Figma
  • UI Design
  • UX Design

Principais funcionalidades

  • Criação de perfil de jogador
  • Formação e gerenciamento de equipes
  • Inscrição em torneios e eventos
  • Painel para organizadores
  • Acompanhamento de rankings
  • Relatórios e histórico de atividades

Portal de Voo

Descrição

O Portal de Voo foi desenvolvido como atividade de finalização de semestre do curso técnico, com o objetivo de praticar conceitos fundamentais de desenvolvimento front-end utilizando HTML e CSS.

O projeto simula um sistema de aviação com tela de login, informações da frota, perfil do piloto e plano de voo ilustrativo, priorizando organização visual, responsividade e estruturação de layouts.

Tecnologias utilizadas

  • HTML5
  • CSS3
  • Flexbox
  • Google Maps (Iframe)

Principais recursos

  • Tela de login
  • Status da frota
  • Perfil do piloto
  • Plano de voo ilustrativo
  • Layout com Flexbox
  • Efeitos de hover e transições
  • Validação de formulário
  • Organização de arquivos e estrutura do projeto

Observação: o plano de voo utiliza um mapa incorporado via iframe apenas para fins ilustrativos.

MovieTracker

Descrição

O MovieTracker é um aplicativo mobile desenvolvido durante a disciplina de Dispositivos Móveis, com foco na criação de interfaces para celular, navegação entre telas e cadastro de informações.

A aplicação permite registrar filmes assistidos, informando título, gênero, data, avaliação, status, plataforma onde foi assistido e comentários pessoais.

Tecnologias utilizadas

  • React Native
  • Expo
  • JavaScript
  • Node.js
  • MySQL
  • API REST

Principais funcionalidades

  • Cadastro de filmes
  • Listagem dos filmes cadastrados
  • Registro de gênero, data e avaliação
  • Status de filme assistido
  • Campo para plataforma onde assistiu
  • Comentários e reviews pessoais
  • Navegação entre telas
  • Integração com API
Quitanda do Tomate Cru

Descrição

Sistema web desenvolvido para gerenciamento de produtos de uma quitanda, permitindo cadastrar, visualizar, editar e excluir registros através de uma interface intuitiva integrada a um banco de dados MySQL.

O projeto foi desenvolvido utilizando Flask no back-end e JavaScript para comunicação assíncrona com a API, aplicando conceitos de CRUD, integração com banco de dados e manipulação dinâmica da interface.

Tecnologias utilizadas

  • Python
  • Flask
  • MySQL
  • HTML5
  • CSS3
  • Bootstrap
  • JavaScript
  • REST API

Principais funcionalidades

  • Cadastro de produtos
  • Listagem dinâmica de registros
  • Edição de produtos
  • Exclusão de registros
  • Validação de campos
  • Mensagens de feedback para o usuário
  • Integração entre Front-end e Back-end
  • Persistência de dados em MySQL

Conceitos aplicados

  • Arquitetura Cliente-Servidor
  • CRUD completo
  • APIs REST
  • Manipulação do DOM
  • Requisições Fetch
  • Integração Flask + MySQL

Principais funcionalidades

  • Listagem de registros
  • Edição de quantidade e preço
  • Exclusão de produtos
  • Gráfico de nível de estoque
  • Exportação de dados para planilha Excel
  • Integração com banco de dados MySQL

Controle de Finanças Pessoais

Descrição

Aplicação web desenvolvida para controle de finanças pessoais, permitindo registrar entradas e saídas financeiras, acompanhar o saldo atual e gerenciar transações de forma simples e organizada.

O sistema utiliza Flask e SQLAlchemy para integração com banco de dados MySQL, possibilitando operações completas de cadastro, edição e exclusão de registros.

Tecnologias utilizadas

  • Python
  • Flask
  • SQLAlchemy
  • PyMySQL
  • MySQL
  • HTML5
  • CSS3
  • JavaScript
  • Jinja2

Principais funcionalidades

  • Cadastro de receitas e despesas
  • Cálculo automático do saldo atual
  • Listagem de transações financeiras
  • Edição de registros
  • Exclusão de transações
  • Persistência dos dados em MySQL
  • Animações e interações com JavaScript
  • Interface responsiva para gerenciamento financeiro
